Converts Kelvin to Fahrenheit.

- Follow the conversion formula `F = (K - 273.15)*9/5 + 32`.

```js
const kelvinToFahrenheit = degrees => (degrees - 273.15)*9/5 + 32;
```

```js
kelvinToFahrenheit(60); // -351.67
```
